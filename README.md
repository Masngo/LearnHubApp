# LearnHub App

LearnHub is a Flutter-based mobile learning platform developed by **Team 8** as the final deliverable of a 4-week internship program. The project demonstrates the design, development, and documentation of a complete mobile application using Flutter, with a strong focus on usability, clean UI design, smooth navigation, and professional software practices.

## Project Overview

The LearnHub App provides users with access to structured learning programs through an intuitive and well-organized interface. Users can authenticate through a secure login flow, navigate a home dashboard, browse available programs, view detailed program information, and submit feedback using interactive forms. The application simulates real-world mobile app behavior through proper state handling, loading indicators, and user feedback messages.

## Features

- User authentication flow (Login, Sign Up, Forgot Password)
- Home dashboard with learning progress indicators
- Program listing with search functionality
- Program details screen with expandable curriculum sections
- Feedback form with validation and submission handling
- Bottom navigation for seamless screen switching
- Loading indicators and success/error feedback
- Logout functionality

## Technologies Used

- Flutter (Dart)
- Material UI
- StatefulWidget for state management
- Git and GitHub for version control
